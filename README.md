# Aplicação web para controle de estoque
Este repositório inclui uma aplicação web que tem finalidade para controlar o estoque de um departamento de TI. Atualmente o software conta com as seguintes ferramentas:
## Front-end
* React.js
* Bootstrap

## Back-end
* Node.js
* Express.js
* Sequelize (ORM)
* Arquitetura MVC
